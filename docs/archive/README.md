# Archive Index

過去の調査メモや旧仕様を保管するアーカイブです。現行ドキュメントに反映済みの内容はこちらで参照できます。

## ディレクトリ構成
- `devlog/` : 調査ログや一時的なトラブル対応メモ  
  - 例: `Realtime-PostgresChanges.md`, `pkce-error-resolution.md`, `realtime-troubleshooting-log.md`
- `legacy-specs/` : 旧仕様・ガイド・チェックリスト  
  - 例: `architecture.md`, `api.md`, `monitoring.md`, `operations-runbook.md`, `multi-device-testing-guide.md`, `google-oauth-*.md`

新たにアーカイブへ移動した場合は、該当ディレクトリ内の README に概要と移動日を追記してください。
